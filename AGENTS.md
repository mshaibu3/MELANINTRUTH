# MelaninTruth AI Agent Instructions

This repository builds a production-grade AI platform for truthful representation of melanin-rich skin.

Never implement beauty filters.
Never lighten, whiten, smooth, reshape, or alter user identity.
All image transformations must be scientifically constrained lighting standardisation.
Every AI output must include confidence, uncertainty, and explanation.
Sensitive images must be treated as high-risk personal data.
No image may be used for training unless explicit opt-in consent exists.
No secrets may be committed.
No raw personal images may be logged.
All API routes must enforce authentication and authorisation unless explicitly public.
All data access must be tenant-safe.
All code must include tests where practical.
Run linting and tests before claiming completion.
If tests cannot run, document the exact reason.
Do not leave TODO comments for core security, privacy, consent, or AI safety controls.

## Definition of Done

* code implemented
* tests added
* migrations added where needed
* documentation updated
* security implications considered
* privacy implications considered
* commands run or failure documented
* no fake production claims
